TAR

tar cvf test.tar . TAR current dir: c=create, v=verbose f=file
tar tvf test.tar list content of TAR file: t=table of content
tar xvf test.tar unTAR all files: x=xtract
tar xvf test.tar file1 unTAR specific file

ZIP

Memdump forensics with volatility

Install:
wget http://downloads.volatilityfoundation.org/releases/2.6/volatility_2.6_mac64_standalone.zip
unzip volatility_2.6_mac64_standalone.zip
cp volatility_2.6_mac64_standalone/volatility_2.6_mac64_standalone /usr/local/bin
ln -s /usr/local/bin/volatility_2.6_mac64_standalone /usr/local/bin/volatility volatility -h

Command reference https://github.com/volatilityfoundation/volatility/wiki/Command-Reference

Image info: volatility -f file.raw imageinfo
Process list: volatility -f file.raw --profile=Win7SP1x86 pslist # or pstree
Hidden process list: psscan
env vars: envars
Command through console shell cmd.exe: cmdscan
File search: filescan
Dump file: dumpfiles -D output/ -Q 0x000000003f22a930 # memory address
Print Reg Key: printkey -K 'RealVNC\\vncserver'
Print Reg Key: printkey -K 'Policy\\Secrets\\DefaultPassword'
LSA dump: lsadump
Screenshot desktops: screenshot -D output/
Clipboard content: clipboard
